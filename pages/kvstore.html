ESP32-Based Time-Series KV Store

1. Overview

This project implements a lightweight time-series KV store on the ESP32, supporting range-based MQTT queries and flash persistence.

2. Hardware Specifications

  • Microcontroller: ESP32-WROOM-32E
  • CPU: Dual-core Xtensa LX6, up to 240 MHz
  • RAM: 520 KB SRAM
  • Flash: 4 MB SPI Flash
  • Connectivity: Wi-Fi (802.11 b/g/n), Bluetooth v4.2

3. System Architecture

The system uses a layered approach for data storage:

  1. RAM Buffer: Holds recent measurements for fast access.
  2. NVS Flash: Stores data persistently once the buffer is full.
  3. Edge Device: (Future) Remote query support when data is not local.

4. Query Workflow

MQTT messages specify a timestamp range, and the ESP32:

  • Parses the query
  • Fetches data from RAM or Flash
  • Sends a JSON response via MQTT

5. Performance

Storage Layer Access Time
RAM Buffer ~100 µs
Flash (NVS) ~1–2 ms
Full Query ~50 ms
