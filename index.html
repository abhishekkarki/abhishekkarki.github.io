Lightweight KV Store

A distributed key-value store for ESP32.

This project implements a time-series key-value (KV) store on an ESP32 module for collecting, storing, and querying temperature measurements. Data is initially buffered in internal RAM for fast access and periodically flushed to non-volatile flash storage (NVS) to ensure persistence. The system supports range queries via MQTT messages, prioritizing reads from the buffer for low-latency access, and falling back to flash or edge-device queries when needed. Query response times are measured and optimized using microsecond-resolution timers. The project integrates seamlessly with an edge device for extended data access and leverages MQTT for lightweight, asynchronous communication.

Infrastructure Setup

Setting a infra using Terraform

This project implements a secure and modular Infrastructure as Code (IaC) setup on different cloud platforms using Terraform. It integrates HashiCorp Vault for secret management, ensuring that cloud credentials are never hardcoded. The infrastructure is organized by environment (dev, stag, prod), follows Terraform best practices, and is fully automated via a Makefile for consistent and repeatable deployments. Vault runs locally in Docker with persistent storage, and Terraform dynamically pulls secrets from Vault during every execution, enabling secure, scalable, and environment-aware cloud provisioning.
